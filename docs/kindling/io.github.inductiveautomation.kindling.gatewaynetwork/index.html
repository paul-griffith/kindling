class GatewayNetworkViewer(tabName: String, tooltip: String, json: String) : ToolPanel

Opens the raw file that contains a gateway network diagram as JSON text. After opening, the user can click a button to open a browser that will show the actual diagram. The diagram is served from html and javascript files copied out to a temp folder on the local file system. The source of the static files is the cytoscape-server IA internal repository, in the 'cytoscape-server.zip' file. Within the zip file, navigate to 'server/static' to view the actual files.
